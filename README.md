elasticsearch-workshops
=======================
Materiały do warsztatóww "Elasticsearch - start bez niespodziedzianek", stworzone pierwotnie na Warsjawę 2013.

Slajdy
------
W folderze *presentation* znajduje się prezentacja, którą można zobaczyć otwierając plik *index.html* w przeglądarce.
Prezentacja zrobiona jest przy pomocoy reveal.js, stąd biorą się w niej dodatkowe pliki licencji.

Data
----
W folderze *data* znajdują się:

*data.txt* - dane dotyczące kilkunastu piw wyprodukowanych przez browary AleBrowar, Artezan, PINTA i Browar Piwoteka.
Wszystkie dane wzięte są z ogolnodostępnych źródeł, a opisy piw są "sloganami" piw ze stron samych browarów, a więc
w rozumieniu prawa autorskiego są prostą informacją prasową, nie podlegającą ochronie. Dane te są w formacie,
który pozwala na bezpośrednie wczytanie ich przy użyciu bulk API Elasticsearcha.

Pozostałe pliki json są przykładami mniej i bardziej skomplikowanych zapytań, które można wykorzystać w trakcie,
w połączeniu ze slajdami.
